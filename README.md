Our own **ray tracing**, 42 project with [Chanmachan](https://github.com/Chanmachan)

### How to use

```
make && ./miniRT <scene.rt>
```

Sample ``.rt`` files are in ``rt_files`` directory.

Or in short,

```
./test.sh
```
